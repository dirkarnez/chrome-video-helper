chrome-video-helper
===================
- [ ] skip to specified time
- [ ] ArrowUp / ArrowDown volume control
- [ ] Loop
  ```js
  document.getElementsByTagName("video" /*"audio"*/)[0].loop = true;
  ```
- [ ] Play video as audio
  ```js
  // replace `<video>` to `<audio>`
  <audio controls src="movie.mp4"></audio>
  ```
